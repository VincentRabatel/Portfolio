// Oh My Food
// Website

Oh my food is a website allowing you to place your order before going to the restaurant.

This service was designed with the "mobile-first" approach and with the wish to be as simple and practical as possible to use.

You can check the code on GitHub and the website itself.

A special attention was paid to the responsive design of the Oh My Food project.
In particular by adopting a mobile first approach, and by building the other breakpoints from this version.
